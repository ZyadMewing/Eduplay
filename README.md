**Deskripsi Singkat** : Eduplay adalah game edukasi anak-anak yang berbasis digital (di ponsel atau tablet) yang dirancang khusus untuk menggabungkan unsur hiburan dan pembelajaran.

**Tujuan utamanya** adalah agar anak-anak tidak membuang waktu saat menggunakan handphone dan juga membuat proses belajar menjadi lebih menyenangkan, interaktif, dan menarik bagi anak-anak. Game edukasi ini memiliki konten yang fokus pada pengembangan keterampilan dan pengetahuan berlogika anak.

**Teknologi yang Digunakan** (framework => flutter, database => penyimpanan lokal, bahasa pemrograman => dart)
